# 原生工具與瀏覽器子代理 (Native Tools & Browser Subagent)

Antigravity 內建了一系列足以與外部物理世界（特別是網頁環境）互動的強大工具。

## 1. 瀏覽器子代理 (Browser Subagent)
當主代理需要深入研究網頁內容或與複雜的 Web UI 互動時，會啟動專門的瀏覽器子代理。
*   **特性**：使用專門為「網頁導航與操作」優化的模型。
*   **視覺警示**：受控的瀏覽器視窗會出現 **藍色框架 (Blue Border)**，表示目前正由 AI 操控。此時應避免手動干預。
*   **具備能力**：點擊、輸入文字、捲動頁面、擷取螢幕快照、分析 DOM、讀取主控台 (Console) 日誌及錄製動作影片。
*   **背景作業**：支援在背景分頁 (Unfocused tabs) 中靜默執行。

## 2. 常用系統工具
代理具備調用基本作業系統介面的能力：
*   **Terminal (終端機)**：執行 Shell 指令（受到沙箱與權限控制）。
*   **File System (檔案系統)**：支援讀取與寫入工作區內的檔案。
*   **Screen Capture (螢幕擷取)**：當代理需要「看見」目前編輯器外部的狀態時，可能會要求擷取螢幕。

## 3. 使用建議
*   **明確目標**：交給瀏覽器子代理任務時，目標越明確（例如：找到文件中關於 X 的定義並翻譯），成功率越高。
*   **觀察日誌**：如果子代理回報錯誤，通常可透過查看該會話錄製的影片或截圖來確認網頁是否發生意外跳窗或阻擋。
